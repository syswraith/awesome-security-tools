# Awesome-security-tools

For CTFs, Reverse Engineering and Problem Solving!

| Tool                                           | Description                                                                                                                                                                                                                                                                                     |
| ---------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Aperi'Solve](https://aperisolve.fr/)          | Uses zsteg, steghide, outguess, exiftool, binwalk, foremost, and strings for deeper steganography analysis. Supports image formats such as `.png`, `.jpg`, `.gif`, `.bmp`, `.jpeg`, `.jfif`, `.jpe`, `.tiff`, and more.                                                                         |
| [CyberChef](https://gchq.github.io/CyberChef/) | A web-based data transformation toolkit with 463+ operations including Base64 decoding, timezone conversion, Teredo IPv6 parsing, hexdump conversion, decompression, shellcode decryption/disassembly, timestamp expansion, multi-type data handling, and dynamic arguments derived from input. |
| [dCode](https://www.dcode.fr/en)               | A collection of 900+ tools to solve ciphers, riddles, puzzles, math problems, geocaching challenges, and daily logic tasks. Free, extensive, and widely used in CTF challenges.                                                                                                                 |
